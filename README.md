Launch

  Required libs 
      <ul>
      <li>Boost</li>
      <li>libpqxx</li>
      <li>openssl</li>
      <li>nlohmann json</li>
      </ul>

  Install
      <ul>
      <li>sudo cmake CMakeLists.txt</li>
      <li>sudo make</li>
      </ul>

  Use
    <ol>
      <li>postgresql_setup.sh - create user, database and tables</li>
      <li>launch Server</li>
      <li>launch Client</li>
    </ol>
